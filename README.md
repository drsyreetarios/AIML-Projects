# AIML Projects Portfolio

This repository contains a collection of Artificial Intelligence and Machine Learning projects completed as part of the AIML curriculum.

## Projects Included:
1. **ML Project**: Core machine learning algorithms and implementations.
2. **NLP RAG Project**: Natural Language Processing using Retrieval-Augmented Generation.
3. **INN ReneWind**: Machine learning applications for renewable wind energy.
4. **EasyVisa**: Predictive modeling for visa approval outcomes.
5. **SuperKart**: End-to-end model deployment and serving.

Each project is located in its respective folder within the `projects/` directory.
